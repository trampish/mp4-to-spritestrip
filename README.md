mp4-to-spritestrip
==================

Handy toolset to create JPG sprite strips from MP4/MOV files in situations where GIF or video files aren't a good solution.



Requirements
-----------

mp4-to-spritestrip uses

* [FFmpeg] - awesome web-based text editor
